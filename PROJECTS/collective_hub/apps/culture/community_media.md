**Problems it solves:**
- Community stories go untold
- No platform for local voices
- Culture doesn't spread

## User Flow

### As Consumer
```
1. Browse content → Find something interesting
2. Listen/watch → Engage with media
3. Interact → Comment, like, share
4. Subscribe → Follow favorite creators
5. Download → Save for offline
```

### As Creator
```
1. Create → Record or upload
2. Edit → Add title, description
3. Publish → Share with community
4. Engage → Respond to comments
5. Track → See views and engagement
```
