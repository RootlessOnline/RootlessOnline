> Tinder for local events. Swipe to find your community.

## Core Purpose

Make it easy for neighbors to discover events and participate in community life.

**Problems it solves:**
- People want to meet neighbors but don't know how
- Community events are hard to discover
- Organizing activities takes effort

---

## Key Features

### Swipe Interface
- Event cards with photo, title, distance
- Swipe right = interested
- Swipe left = skip
- Swipe up = more info

### Event Cards Show
- Event title
- Date and time
- Distance from user
- Number interested
- Category icon
- Quick preview

### Event Types
- 🌱 Gardening workdays
- 🍲 Neighborhood dinners
- 🎬 Outdoor movie nights
- 🧹 Park cleanups
- 🧑‍🏫 Skill workshops
- 🎲 Board game nights
- 📚 Book clubs
- 🏃 Sports activities

### Group Formation
- Events activate when enough people join
- Example: "Plant trees in park" needs 6 people
- When 6 swipe right → event confirmed

### Event Creation
- Anyone can create
- Set minimum participants
- Add photos, description
- Choose location

### Temporary Chats
- Each event gets a chat
- Coordinate with other participants
- Chat closes after event

### Privacy First
- No public profiles unless opted in
- See who else joined (optional)
- Control visibility

---

## User Flow

```
1. Open app → See swipeable event cards
2. Swipe right on interesting events
3. When enough people → Event confirmed
4. Join temporary chat
5. Attend event
6. Log participation (credits)
```

```
├── created_by (user)

├── user_id
├── event_id
├── status (interested/confirmed/attended)
└── credits_earned
```
