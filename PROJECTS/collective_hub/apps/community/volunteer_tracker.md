# Volunteer Tracker

> Track community contributions and earn credits.

---

## App Information

| Property | Value |
|----------|-------|
| Category | Community |
| Priority | Phase 2 |
| Platform | Collective Hub (module) |

---

## Core Purpose

Recognize and reward community contributions.

**Problems it solves:**
- Volunteers feel unappreciated
- No record of contributions
- Hard to track community involvement

---

## Key Features

### Contribution Logging
- Hours volunteered
- Tasks completed
- Events attended
- Skills shared

### Credit System
- Earn credits per hour
- Different rates for different tasks
- Community-verified contributions

### Profile Building
- Contribution history
- Skills offered
- Badges earned
- Community standing

### Recognition
- Top contributor highlights
- Milestone badges
- Community thanks

### Verification
- Event organizer confirms attendance
- Photo evidence
- Peer verification

---

## Credit Types

| Activity | Credits |
|----------|---------|
| Garden workday | 10/hour |
| Teaching a skill | 15/hour |
| Organizing event | 20/event |
| Tool sharing | 5/use |
| Knowledge contribution | 10/guide |

---

## User Flow

```
1. Volunteer at event → Log participation
2. Organizer verifies → Credits awarded
3. Build profile → See contribution history
4. Earn badges → Recognition milestones
5. Use credits → Trade for services (optional)
```

---

## Integration with Hub

- **Neighbor Connection** → Auto-log event attendance
- **Garden Locator** → Garden volunteer hours
- **Tool Library** → Lending/borrowing credits
- **How-To Hub** → Teaching credits

## Data Model

```
Contribution
├── id

├── type (volunteering/teaching/sharing/etc)
├── hours
├── credits_earned
├── description
├── verified_by (user_id)
├── date
└── related_event_id (optional)

UserCredits
├── user_id
├── total_credits
├── total_hours
├── badges[]
└── rank
```
