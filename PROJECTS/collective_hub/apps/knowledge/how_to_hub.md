## Core Purpose

Create a free library of practical skills shared by the community.

**Problems it solves:**
- Useful knowledge is scattered
- Many people want to learn practical skills
- Community knowledge gets lost

## User Flow

### As Learner
```
1. Search topic → Find what you want to learn
2. Open guide → Read/watch content
3. Follow steps → Complete tutorial
4. Save for later → Bookmark helpful guides
5. Ask questions → Comment section
6. Rate → Help others find best guides
```

### As Contributor
```
1. Create guide → Title, category, difficulty
2. Add content → Steps, photos, videos
3. List requirements → Tools, materials, time
4. Publish → Community can find it
5. Respond to questions → Help learners
6. Update → Improve based on feedback
```
