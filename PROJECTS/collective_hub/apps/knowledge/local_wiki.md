| Property | Value |
|----------|-------|

---

## Core Purpose

Allow communities to document local knowledge and resources.

**Problems it solves:**
- Important local information is hard to find
- Community knowledge disappears
- Projects lack shared documentation

---

## Key Features

### Wiki Pages
- Editable by community
- Version history
- Categories and tags
- Searchable

### Content Types
- 📖 Guides and how-tos
- 📍 Local resources
- 🏛️ Community history
- 📋 Project documentation
- 🗺️ Neighborhood info
- ⚖️ Local policies

### Organization
- Categories
- Tags
- Collections
- Featured pages

### Collaboration
- Multiple editors
- Discussion pages
- Change proposals
- Moderation queue

### Quality
- Community verification
- Expert badges
- Citation support
- Flag outdated content

---

## User Flow

```
1. Search topic → Find existing pages
2. Read page → Get information
3. Edit/Improve → Add knowledge
4. Submit change → For review
5. Community approves → Page updated
```

---

## Page Structure

```markdown
# Page Title

## Overview
Brief summary of the topic.

## Details
Main content area.

## Related Pages
Links to connected topics.

## External Links
References and sources.

## Discussion
Community comments.

## History
Edit log.
```

```

├── id
├── page_id
├── editor_id
├── previous_content

├── timestamp
```
