# Garden Locator

> Find community gardens and growing spaces near you.

---

## App Information

| Property | Value |
|----------|-------|
| Category | Food & Growing |
| Priority | ⭐ MVP |
| Platform | Collective Hub (module) |

---

## Core Purpose

Help people find places to grow food, learn gardening, or participate in community gardens nearby.

**Problems it solves:**
- People want to garden but don't know where
- Community gardens are hard to discover
- Many gardens need volunteers but lack visibility

---

## Key Features

### Map Interface
- Interactive map with garden pins
- Filter by type:
  - Community garden
  - Educational garden
  - Private garden (optional visibility)
  - Farm
  - Edible public space

### Garden Profiles
- Name and description
- Photos
- Location and hours
- What's growing
- Volunteer opportunities
- Contact info

### Discovery
- Filter by distance
- Filter by type
- "Looking for volunteers"
- Events at garden

### Contribution
- Add new gardens
- Update garden info
- Share photos

---

## User Flow

```
1. Open map → See gardens nearby
2. Tap garden → View profile
3. Join event / Volunteer / Follow
4. Get updates from garden
5. Add your own garden
```

## Integration with Hub

- **Neighbor Connection** → Events at gardens appear in event feed
- **How-To Hub** → Guides linked to garden profiles
- **Tool Library** → Tools available at each garden
- **Volunteer Tracker** → Hours logged per garden

## Data Model

```
Garden
├── id
├── name
├── description
├── location (lat/lng)
├── type (community/private/farm/etc)
├── photos[]
├── plants[]
├── volunteer_status (open/closed)
├── events[]
├── contact
└── created_by (user)
```
