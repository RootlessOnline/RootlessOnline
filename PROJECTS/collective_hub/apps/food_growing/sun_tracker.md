| Property | Value |

---

## Core Purpose

Visualize sunlight patterns across the day and year for better garden planning.

**Problems it solves:**
- Don't know where sun hits garden
- Wrong plants in wrong light
- Seasonal sun changes unexpected

---

## Key Features

### Sun Angle Visualization
- Curved sunrise/sunset indicator
- Current sun position
- Shadow mapping

### Seasonal Tracking
- How sun changes through year
- Summer vs winter sun paths
- Planning for all seasons

### Garden Mapping
- Mark obstacles (trees, buildings)
- Calculate shade zones
- Full sun / partial shade / full shade areas

### Plant Matching
- Sun requirements per plant
- Best placement suggestions
- Seasonal adjustments

---

## User Flow

```
1. Enter location → Auto-detect or manual
2. Map garden → Draw beds, mark obstacles
3. See sun patterns → Real-time visualization
4. Get recommendations → Where to plant what
5. Plan seasons → Sun throughout the year
```

## Integration with Hub

## Data Model
