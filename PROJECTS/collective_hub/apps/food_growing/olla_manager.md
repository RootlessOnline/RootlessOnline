# Olla Manager

> Track and manage olla irrigation systems.

| Property | Value |
|----------|-------|
| Category | Food & Growing |

---

## Core Purpose

Manage clay pot irrigation systems for efficient water use.

**Problems it solves:**
- Forget to refill ollas
- Don't know water usage
- Unclear which plants need how much water

---

## Key Features

### Olla Inventory
- Track all ollas in garden
- Size and capacity
- Location mapping
- Installation date

### Refill Reminders
- Notification when to refill
- Based on weather and plant needs
- Customizable frequency

### Water Tracking
- Water usage per olla
- Comparison to traditional irrigation
- Savings calculator (60-70%)

### Plant Matching
- Which plants near each olla
- Water needs per plant
- Optimal olla size recommendations

### DIY Guides
- How to make ollas
- Installation tutorials
- Maintenance tips

---

## User Flow

```
1. Add olla → Size, location, linked plants
2. Set schedule → Refill frequency
3. Get reminders → Notification to refill
4. Log refills → Track water usage
5. View stats → Water savings over time
```
