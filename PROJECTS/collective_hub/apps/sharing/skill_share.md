## Core Purpose

Connect people who have skills with people who want to learn them.

**Problems it solves:**
- Skills are gatekept
- Learning is expensive
- Teachers are hard to find
- Knowledge stays isolated

---

## Key Features

### Skill Listings
- Browse by category
- Skill level (beginner/intermediate/advanced)
- Teaching format (in-person/online)
- Availability

### Categories
- 🌱 Gardening & farming
- 🔨 Building & repairs
- 🍳 Cooking & preserving
- 🧵 Crafts & making
- 💻 Technology
- 🎨 Arts & creative
- 🧘 Health & wellness
- 📚 Languages

### Teaching
- Offer to teach
- Set your terms
- One-on-one or group
- Free or barter or paid

### Learning
- Search skills
- Find teachers nearby
- Request a skill
- Schedule sessions

### Exchange
- Skill swap (I teach X, you teach Y)
- Time bank integration
- Community credits
- Direct payment (optional)

---

## User Flow

### As Teacher
```
1. Create profile → List skills you can teach
2. Set availability → When/how you teach
3. Receive requests → Review learners
4. Accept/decline → Confirm sessions
5. Teach → Share knowledge
6. Get rated → Build reputation
```

### As Learner
```
1. Search skill → Find what you want to learn
2. Filter results → Distance, format, price
3. Request session → Message teacher
4. Confirm → Agree on time/place
5. Learn → Attend session
6. Rate → Give feedback
```

## Data Model

```

├── id

├── category
├── description

├── format (in-person/online/both)

├── availability
└── rating

SkillRequest
├── id
├── learner_id
├── skill_name
├── description
├── desired_format
└── open/closed

Session
├── id
├── skill_id
├── teacher_id
├── learner_id
├── datetime
├── location
├── status (scheduled/completed/cancelled)
└── ratings[]
```
