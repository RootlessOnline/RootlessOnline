## Core Purpose

Connect local skills with local needs for paid work and services.

**Problems it solves:**
- Hard to find reliable local help
- Skills underutilized
- Money leaves community
- Job matching is inefficient

---

## Data Model

```
Service
├── id
├── provider_id
├── title
├── description
├── category
├── rate_type (hourly/fixed)
├── rate_amount
├── availability
├── portfolio[]
└── rating

Job
├── id
├── poster_id
├── title
├── description
├── category
├── budget
├── timeline
├── location
├── proposals[]
├── status (open/in_progress/completed)
└── selected_provider
```
