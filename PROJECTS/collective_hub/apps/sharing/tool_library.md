## Core Purpose

Enable tool sharing so people don't need to buy what they rarely use.

**Problems it solves:**
- Tools are expensive
- People own duplicates
- Storage space wasted
- Not everyone can afford tools

---

## Key Features

### Tool Inventory
- Browse available tools
- Filter by category
- See availability status
- View tool condition

### Categories
- 🌱 Gardening (shovels, rakes, tillers)
- 🔨 Construction (drills, saws, levels)
- 🚗 Automotive (jacks, specialty tools)
- 🎨 DIY/Craft (sewing machines, art supplies)
- 🏠 Household (ladders, pressure washers)
- 🔧 Electronics (multimeters, soldering)

### Tool Details
- Photos
- Description and specs
- Owner information
- Condition rating
- Instructions/guides

### Borrowing System
- Request tool
- Set pickup/return dates
- Message owner
- Confirm pickup
- Return confirmation

### Trust System
- User ratings
- Return history
- Deposit/waiver options
- Damage reporting

### Location
- Map view of tools
- Distance filtering
- Pickup arrangements

## User Flow

### Borrower
```
1. Search for tool → Find what you need
2. Check availability → See if it's free
3. Request borrow → Set dates, message owner
4. Arrange pickup → Coordinate time/place
5. Use tool → Follow guidelines
6. Return tool → Confirm return, rate experience
```

### Lender
```
1. Add tool → Photos, description
2. Set availability → When you can lend
3. Review requests → Accept/decline
4. Hand off → Note condition
5. Receive return → Check condition
6. Rate borrower → Build trust
```

## Data Model

```
Tool
├── id
├── name
├── description
├── category
├── owner_id (user)
├── location
├── photos[]
├── condition (new/good/fair/needs_repair)
├── availability_rules
├── deposit_required (optional)
├── instructions
└── current_status (available/borrowed/maintenance)

Borrow
├── id
├── tool_id
├── borrower_id
├── lender_id
├── start_date
├── end_date
├── status (requested/active/returned/overdue)
├── pickup_location
├── condition_before
├── condition_after
└── ratings (both sides)
```
