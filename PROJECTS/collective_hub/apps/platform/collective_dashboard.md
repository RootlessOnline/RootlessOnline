## Core Purpose

The central entry point for the entire Collective ecosystem.

**Problems it solves:**
- Users need one place to start
- Activity across apps needs visibility
- Community pulse needs a home

---

## Key Features

### Home Screen
- Activity feed for your area
- Upcoming events
- New guides and tutorials
- Community announcements
- Garden updates nearby

### App Launcher
- Grid of available apps
- Recently used
- Favorites
- Quick actions

### Map View
- Central map showing:
  - Gardens
  - Events
  - Tool libraries
  - Community spaces
- Filter by type
- Search locations

### Community Tab
- Your neighborhood
- Active projects
- Discussion threads
- Member directory

### Profile
- Your contributions
- Skills and interests
- Saved items
- Settings
