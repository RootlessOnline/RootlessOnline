> Step-by-step instructions to get The Collective running.
 Harley: Complete each step in order. Don't skip.

## PHASE 0: ESSENTIALS (Day 1)

### Step 0.1: Write Purpose
- [ ] Open `CONSTITUTION.md`
- [ ] Write ONE sentence under PURPOSE
- [ ] Example: "The Collective exists to build regenerative communities through gardens, shared resources, and participatory governance."
- [ ] Fallback: Z proposes default → Harley approves

### Step 0.2: Write Principles
- [ ] Under CONSTITUTION.md PRINCIPLES section
- [ ] Write 3-5 core principles
- [ ] These guide every decision
- [ ] Example:
  - Principle 1: Community First
  - Principle 2: Regenerative Systems
  - Principle 3: Distributed Power

### Step 0.3: Create START.md
- [ ] Z will do this automatically
- [ ] Links to your purpose + principles
- [ ] Becomes the entry point for all AIs

### Step 0.4: First Decision
- [ ] Open `STATE/decision_log.md`
- [ ] Create ADR-001
- [ ] Example decision: "Garden business = Phase 1"
- [ ] Z can help draft this

### Step 0.5: Define Your Intuition
- [ ] Open `EXPERTISE/harley_intuition/README.md`
- [ ] Write 3-5 things you know that are hard to explain
- [ ] Examples:
  - How you know a plant will thrive
  - How you read people
  - What makes a good garden design
