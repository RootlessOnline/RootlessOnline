## Bootstrap Principles

1. **Start Small:** One garden, one tool, one community
2. **Document Everything:** If it's not written down, it didn't happen
3. **Learn Fast:** Try things, fail small, iterate
4. **Build Momentum:** Small wins create energy
5. **Stay Regenerative:** Every action should build capacity

---

## Emergency Bootstrap

If something breaks or goes wrong:

1. **Check current_state.md** — Is the issue documented?
2. **Check decision_log.md** — Has this happened before?
3. **Check conflict_playbook.md** — Is there a protocol?
4. **Escalate to Harley** — If none of the above help
