## What Is This?

The Collective is an AI-human collaboration system working to build:
1. **Garden Business** — Olla irrigation, permaculture, therapeutic horticulture
2. **Collective App** — Platform and tools for community coordination
3. **Community Tools** — Tool library, neighborhood connection
4. **Political Party** — Netherlands-based civic engagement
